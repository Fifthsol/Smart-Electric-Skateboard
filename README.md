# Smart-Electric-Skateboard


This is my smart electric skateboard that I'm building for hack club.

To put simply the goal of this project is to have an electric skateboard that is fully integrated with bluetooth including Strava posting capabilities. 

It will feature an OLED display, a gyroscope/accelerometer module, a spark key that works as a circuit finisher for the entire board, a buzzer, and will run off of an ESP32 and an Arduino Nano(the nano is purely for datalogging to avoid clogging the ESP32).

This will be compatible with virtually all motors as long as it's a normal electric skateboard voltage :).
